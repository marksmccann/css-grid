Gutters

Since grid.css has been written with LESS, you can define the size of the gutters (space between grid-blocks).

// small screens +
@gutter-2: 1.25rem; // 20px
@gutter-3: .9375rem; // 15px
@gutter-4: .625rem; // 10px
@gutter-5: .5rem; // 8px
@gutter-6: .5rem; // 8px
// medium screens +
@gutter-2-medium: 1.875rem; // 30px
@gutter-3-medium: 1.25rem; // 20px
@gutter-4-medium: .9375rem; // 15px
@gutter-5-medium: .625rem; // 10px
@gutter-6-medium: .625rem; // 10px
// large screens +
@gutter-2-large: 2.8125rem; // 45px
@gutter-3-large: 1.875rem; // 30px
@gutter-4-large: 1.25rem; // 20px
@gutter-5-large: .9375rem; // 15px
@gutter-6-large: .9375rem; // 15px
