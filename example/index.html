grid.css

A responsive, versatile grid system

 
 
 
 
 
 
 
 
 
 
 
 
 
 

Basics

The markup structure couldn't be any simplier; a container with the defined number of columns desired and a child for each.

<div class="grid-x"> <!-- replace the "x" with the # of columns desired (1-6) -->
    <div class="grid-block"></div>
    <div class="grid-block"></div>
</div>

Responsive

Control the number of columns for each screen size with just a few classes.

<!-- small screens + -->
<div class="grid-x"> ... </div>

<!-- medium screens + -->
<div class="grid-x-md"> ... </div>

<!-- large screens only -->
<div class="grid-x-lg"> ... </div>

Mix and match the classes above. Here are a few examples:

<div class="grid-3-md grid-6-lg"> ... </div>
 
 
 
 
 
 
<div class="grid-2 grid-3-md grid-5-lg"> ... </div>

Gutters

Since grid.css has been written with LESS, you can define the size of the gutters (space between grid-blocks).

// small screens +
@gutter-2: 1.25rem; // 20px
@gutter-3: .9375rem; // 15px
@gutter-4: .625rem; // 10px
@gutter-5: .5rem; // 8px
@gutter-6: .5rem; // 8px
// medium screens +
@gutter-2-md: 1.875rem; // 30px
@gutter-3-md: 1.25rem; // 20px
@gutter-4-md: .9375rem; // 15px
@gutter-5-md: .625rem; // 10px
@gutter-6-md: .625rem; // 10px
// large screens +
@gutter-2-lg: 2.8125rem; // 45px
@gutter-3-lg: 1.875rem; // 30px
@gutter-4-lg: 1.25rem; // 20px
@gutter-5-lg: .9375rem; // 15px
@gutter-6-lg: .9375rem; // 15px
